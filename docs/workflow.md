# AI Agents Factory Workflow

## Objetivo
Definir como os agentes operam de forma integrada para gerar produtos digitais.

## Pipeline dos agentes

1. Research Agent
    - Pesquisa oportunidades e conteúdos
    - Gera dados brutos para processamento

2. Builder Agent
    - Recebe dados do Research Agent
    - Cria produtos ou protótipos iniciais

3. QA Agent
    - Testa qualidade e valida informações
    - Corrige erros do Builder Agent

4. Publisher Agent
    - Prepara e publica o conteúdo final
    - Garante entrega nos canais de distribuição

5. Growth Agent
    - Monitora métricas e tráfego
    - Sugere melhorias para Research/Builder
