   AI Agents Factory Architeture

Este projeto descreve uma arquitetura de agentes de IA operando em um ambiente seguro utilizando Qubes OS.

   Objetivo

Criar múltpilos agentes de IA operando máquinas virtuais separadas e anônimas para produzir e gerenciar produtos digitais.

   Estrutura inicial

Command Center VM
Research Agent VM
Builder Agent VM
QA Agent VM
Publisher Agent VM
Growth Agent VM

Cada agente opera em uma VM isolada utilizando Qubes OS para segurança operacional.
